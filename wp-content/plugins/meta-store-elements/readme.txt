A companion plugin for Meta Store WordPress Theme.

== Description ==
Meta Store Elements is a companion plugin for Meta Store theme. The plugin contains the basic elementor elements.

== Installation ==
1. Make sure you have installed and activated the \'Elementor\' plugin before installing the plugin then,
2. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
3. Activate the plugin through the \'Plugins\' menu in WordPress.
4. Go to Pages > Add New
5. Press the \'Edit with Elementor\' button.
6. Now you can drag and drop widgets under `Meta Store Elements` from the left panel onto the content area, as well as add new sections and columns that make up the page structure.
